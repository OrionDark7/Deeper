Welcome to Deeper!

Deeper is a 2D Video Game Created by @OrionDark7 with Python & Pygame. It is a game set in the Depths of an unknown world, where you can Build, Dig, and Explore freely. Version 0.1.1 is avalible for download, READ INSTRUCTIONS BELOW.

How to Download:

Dont automatticaly hit the Donwload button, there are specific things to download first, or the game won't run:

Step 1. Download Python 2.7

Step 2. Download Pygame 1.9

Step 3. Open and finish the installation process for Python & Pygame. (Don't worry, they aren't viruses, I use both all the time.)

Step 4. Hit the .zip Download button.

Step 5. Extract the .zip file.

Step 6. Open the file Deeper.py, if everything is installed correctly, the game should run. Enjoy!

Welcome to Deeper!

Deeper is a 2D Video Game Created by @OrionDark7 with Python & Pygame. It is a game set in the Depths of an unknown world, where you can Build, Dig, and Explore freely. Version 0.1.2 is avalible for download on Windows 32 bit and higher.

Download for Windows

To run on Windows...

Extract the .zip file, and find 'Deeper.exe' and open it. Enjoy!

Welcome to Deeper!

Deeper is a 2D Video Game Created by @OrionDark7 with Python & Pygame. It is a game set in the Depths of an unknown world, where you can Build, Dig, and Explore freely. Version 0.1.4 is Avalible for Download for Windows, but has no set release date for macOS and OS X. The Python source code for 0.1.4 can also be downloaded here.
